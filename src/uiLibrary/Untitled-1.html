Type scale

A type scale is part of a consistent visual rhythm.

A type scale ensures all the text in the product is part of a set of harmonious proportions. We've applied some tooling to the type scale to help site maintainers and content contributors focus on their work without worrying about font sizes. We’re using a slightly modified one-fifth scale from a base 16px for our fonts:

Sometimes you need a different font size while keeping the html semantic. Luckily classes do not care about the type of tag they are styling. The typescale is accessible through the classes f0 through f6 regardless of the html it is styling.

Sample

Headline

markup

<h1>Headline</h1> | 
<h2 class='f5'>Headline</h1> | 
<p class='f5 serif'>Headline</p>

styles

font-size: 3em;
@media screen and (min-width: $tablet-width){
  font-size: 5em
}

Buttons

Buttons are the primary way of interacting with the site.

It’s important to keep buttons clear and consistent. A user should be able to navigate with as little cognitive overhead as possible. We do this by standardizing button styles and their implementations.
